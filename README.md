## ✨ 功能亮点

- **多角色协同评审**：同时调用广告专家、法律合规官、用户体验师等不同视角
- **双模态支持**：可评审 **纯文本** 或 **带图文案（自动识别图片中的文字）**
- **完全隐私保护**：你的 DashScope API Key 只在本地浏览器使用，**不会发送到任何服务器**
- **开箱即用**：只需一个免费 API Key，5 分钟启动
- **高度可扩展**：通过 YAML 文件轻松添加新角色
